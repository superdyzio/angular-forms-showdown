Signal Forms

Krótka charakterystyka

  • Wykorzystują sygnały (signals) do reaktywnego zarządzania stanem formularza
  • Deklaratywna, czytelna składnia integrująca walidację i zależności między polami
  • Bardziej granularna, precyzyjna re-aktywność i aktualizacje UI
  • Ułatwiają obsługę dynamicznych, złożonych i dużych formularzy
  • Optymalizują wydajność i minimalizują niepotrzebne zmiany detekcji
  • Nowoczesne podejście, wymagające poznania nowego API (Angular Signals)
